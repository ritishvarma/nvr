# NVR Constructions Website

A professional construction company website showcasing services, projects, and contact information for NVR Constructions. Built with React and Vite.

## Running the Project Locally

Follow these steps to run the NVR project on your local machine:

### Prerequisites

- [Node.js](https://nodejs.org/) (which includes `npm`) must be installed on your computer.

### Steps

1. **Open the Terminal**
   Navigate to the project folder in your command line:
   ```bash
   cd nvr
   ```

2. **Install Dependencies**
   If you have not installed the dependencies yet, run:
   ```bash
   npm install
   ```
   *(Note: The `node_modules` folder might already be present if you installed them previously).*

3. **Start the Development Server**
   Run the development script to start the local server:
   ```bash
   npm run dev
   ```

4. **View in the Browser**
   After running the start command, the terminal will display a local URL (typically `http://localhost:5173/`).
   Open your preferred web browser and go to that address.

## Scripts Available

- `npm run dev`: Starts the development server.
- `npm run build`: Builds the app for production.
- `npm run preview`: Locally preview the production build.
- `npm run lint`: Run ESLint to catch errors in the code.
